# SLU2021_EutrophicationMitigationExperiment

Analysis (R script) of "Water darkening modulates the efficiency of coastal eutrophication mitigation in a mesocosm experiment" accepted in the Canadian Journal of Fisheries and Aquatic Sciences.

All data are available on https://doi.org/10.5281/zenodo.14534317 ; with the metadata logged in the file "SLU2021_MitigationExperiment_metadata.ods"
